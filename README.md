# Mobile Programming 1 Exercise Solution

## معرفی پروژه

این مخزن مجموعه ای از مثال ها، تمرین ها و پاسخ های آموزشی درس «برنامه نویسی موبایل 1» است که با تمرکز بر مفاهیم پایه و شیءگرایی در زبان Java سازمان دهی شده است. محتوای پروژه به صورت سشن بندی شده ارائه شده تا دانشجویان، مدرسین و علاقه مندان بتوانند مسیر یادگیری را به شکل مرحله ای و منظم دنبال کنند.

هدف اصلی این پروژه، ارائه نمونه کدهای ساده، قابل فهم و آموزشی برای مفاهیمی مانند متغیرها، کپسوله سازی، وراثت، پیاده سازی Interface، بازنویسی و بارگذاری متدها، اعضای ایستا و کلاس های انتزاعی است. این مخزن همچنین شامل فایل های تمرینی و پاسخ برخی تمرین ها برای استفاده در فرآیند آموزش و تمرین فردی است.

## اهداف اصلی

1. ارائه مثال های آموزشی و دسته بندی شده برای مفاهیم پایه Java
2. کمک به درک بهتر مباحث شیءگرایی در قالب پروژه های کوچک و مستقل
3. فراهم کردن مرجع منظم برای مرور کدهای کلاس و تمرین های عملی
4. تسهیل دسترسی دانشجویان به پاسخ تمرین ها و نمونه پیاده سازی ها

## ویژگی های کلیدی

- ساختار سشن بندی شده بر اساس موضوعات آموزشی
- وجود چندین پروژه نمونه مستقل برای هر مبحث
- استفاده از مثال های ساده و مناسب برای آموزش گام به گام
- تفکیک فایل های سورس در پوشه `src` برای هر پروژه
- وجود فایل های تمرینی در قالب `pdf` و `docx`

## فناوری های استفاده شده

- **زبان برنامه نویسی:** Java
- **محیط توسعه:** IntelliJ IDEA
- **نوع پروژه ها:** پروژه های آموزشی ساده مبتنی بر Java SE
- **مدیریت نسخه:** Git و GitHub

توسعه، سازمان دهی و نگهداری کدهای این مخزن با استفاده از **IDE IntelliJ IDEA** انجام شده است و فایل های تنظیمات مرتبط با این محیط نیز در بخش های مختلف پروژه قابل مشاهده هستند.

## ساختار کلی مخزن

پروژه به صورت موضوعی و بر اساس سشن های آموزشی دسته بندی شده است. برخی از بخش های اصلی مخزن عبارت اند از:

- `Variables (session 1)`
- `Encapsulation (session 2)`
- `Extends (session 3)`
- `Implements (session 3)`
- `Overloading (session 3)`
- `Overriding (session 3)`
- `Static (session 4)`
- `Abstract (session 4)`
- `Exercises`

هر پوشه اصلی ممکن است شامل یک یا چند پروژه نمونه مستقل باشد و هر پروژه معمولا فایل های تنظیمات IntelliJ و یک پوشه `src` مخصوص به خود را دارد.

## راهنمای دسترسی به کدهای پروژه

برای مشاهده کامل فایل های سورس پروژه در GitHub، مراحل زیر را دنبال کنید:

1. ابتدا وارد **پوشه اصلی پروژه** یا همان شاخه مورد نظر خود در مخزن شوید.
2. سپس **پوشه پروژه آموزشی مرتبط** با مبحث مورد نظر را باز کنید.
3. در مرحله بعد، **پوشه `src`** را باز کنید.
4. تمامی فایل های سورس، کلاس ها و کدهای توسعه دهنده درون پوشه `src` قرار دارند.

نکته مهم: در این مخزن، هر مثال یا پروژه آموزشی معمولا ساختار مستقل خود را دارد؛ بنابراین برای دسترسی به کدهای هر بخش، باید وارد همان زیرپوشه شده و سپس پوشه `src` آن را باز کنید. به بیان ساده، بازدیدکنندگان GitHub برای مشاهده کامل سورس ها باید ابتدا به پوشه اصلی مبحث مراجعه کرده و سپس وارد پوشه `src` پروژه مربوطه شوند.

## پیش نیازها

برای مشاهده، اجرا یا توسعه پروژه، موارد زیر پیشنهاد می شود:

- نصب Java Development Kit (JDK) نسخه 8 یا بالاتر
- نصب IntelliJ IDEA
- نصب Git برای دریافت یا به روزرسانی مخزن
- آشنایی مقدماتی با Java و ساختار پروژه های آموزشی

## نصب و راه اندازی

1. مخزن را Clone کنید:

```bash
git clone https://github.com/MahdiGhodsikhah/Mobile-Programming-1-Exercise-Solution.git
```

2. وارد پوشه پروژه شوید:

```bash
cd Mobile-Programming-1-Exercise-Solution
```

3. پروژه یا مثال مورد نظر خود را در IntelliJ IDEA باز کنید.

4. در هر مبحث، زیرپوشه پروژه مورد نظر را انتخاب کرده و فایل های آن را Load کنید.

5. پس از باز کردن پروژه، پوشه `src` را بررسی کنید تا به فایل های کد دسترسی داشته باشید.

## نحوه اجرای پروژه

با توجه به اینکه این مخزن شامل چندین مثال مستقل است، اجرای پروژه ها به صورت موردی انجام می شود:

1. IntelliJ IDEA را اجرا کنید.
2. یکی از زیرپوشه های آموزشی پروژه را باز کنید؛ برای مثال:
   - `Variables (session 1)/Refrens AND primitive`
   - `Encapsulation (session 2)/Full Example with Encapsulation`
   - `Extends (session 3)/extends 2`
3. فایل `Main.java` موجود در پوشه `src` را باز کنید.
4. پروژه را از طریق گزینه Run در IntelliJ IDEA اجرا کنید.

در اغلب مثال ها، نقطه شروع برنامه فایل `Main.java` است.
